# BoxMessage

## Base

This is the most basic version of the component and its default appearance,
normally used with status info.

<Canvas of={Stories.Basic} />

## Elements

Below are some elements that can be added to the base component:

### Content

It is possible to add content in order to add more information related to the
notification:

<Canvas of={Stories.Content} />

### Close button

It is possible to add a remove case icon button
([see IconButton](https://engineering.pages.devotools.com/libraries/design/dali/?path=/docs/components-iconbutton-html-css--page))
to close the BoxMessage:

<Canvas of={Stories.CloseButton} />

### Buttons

It is possible to add some buttons at the end of the BoxMessage to carry out
whatever action is needed:

<Canvas of={Stories.Buttons} />

## Modifiers

This section includes the variants or modifiers that change the appearance of
the base component.

### Color scheme

The BoxMessage color scheme only affects to its status.

#### Status

There are five status: `error`, `help`, `info`, `success` and `warning`.

<Canvas of={Stories.Status} />
